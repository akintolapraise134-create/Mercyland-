# Mercyland School Website

This is a Django-based website for a school. Initial structure generated.

## 🚀 How to run locally

```bash
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt
python manage.py runserver
```

## Project Structure
- Django 5.x+ base project (see `manage.py` and settings)
- To be expanded with apps for students, staff, announcements, contact, etc.
